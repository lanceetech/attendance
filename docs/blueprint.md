# **App Name**: ClassSync

## Core Features:

- Schedule Conflict Resolution: Automatically detect timetable clashes based on lecturer availability, student class times, and room usage, and propose alternative time slots using a tool.
- User Role Management: Admin role to upload units, assign lecturers, schedule classes; Lecturer and Student roles for viewing timetables.
- Timetable Generation & Display: Generate and display personalized timetables for lecturers and students with real-time updates.
- Timetable Download: Allow lecturers and students to download their timetables for offline access.
- Feedback System: Enable lecturers and students to report timetable errors or inconsistencies directly within the system.
- QR Code Attendance: Implement a QR code system for attendance tracking.
- Classroom Status Tracking: Monitor and display the current status of classrooms.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and organization.
- Background color: Light gray (#F0F4F8), providing a clean and calm backdrop.
- Accent color: Orange (#FFB74D) to highlight important actions and notifications.
- Font pairing: 'Inter' for body text (sans-serif) providing a clean and readable interface, and 'PT Sans' for headers (sans-serif) offering a touch of warmth.
- Use clear and intuitive icons to represent different classes, tasks, and notifications.
- Employ a clean and structured layout to present timetable information clearly.
- Use subtle animations for transitions and updates to enhance user experience.
