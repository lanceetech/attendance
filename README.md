# ClassSync: Smart Academic Scheduler & Attendance System

This is a Next.js application built in Firebase Studio that provides a platform for managing academic schedules, tracking student attendance, and resolving timetable conflicts.

To get started, explore the different user roles by signing in as an administrator, lecturer, or student.

- **Admin Login**: Use the "Sign In as Administrator" button on the login page.
- **Student/Lecturer**: Create a new account using the signup form.

## Tech Stack

This project is built with a modern, robust, and scalable tech stack:

- **Framework**: [Next.js](https://nextjs.org/) (using the App Router) for server-rendered React applications.
- **Language**: [TypeScript](https://www.typescriptlang.org/) for static typing and improved code quality.
- **UI Library**: [React](https://react.dev/) for building interactive user interfaces.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for a utility-first styling approach.
- **UI Components**: [ShadCN UI](https://ui.shadcn.com/) for a pre-built, accessible, and customizable component library.
- **Database & Authentication**: [Firebase](https://firebase.google.com/) for real-time data storage with Firestore and user management with Firebase Authentication.
- **Generative AI**: [Genkit](https://firebase.google.com/docs/genkit) (integrated with Gemini) for AI-powered features like schedule conflict resolution.
- **State Management**: React Hooks (`useState`, `useContext`, `useReducer`) for managing component and application state.
- **Form Handling**: [React Hook Form](https://react-hook-form.com/) and [Zod](https://zod.dev/) for efficient and type-safe form validation.
- **Icons**: [Lucide React](https://lucide.dev/) for a comprehensive and lightweight icon set.
